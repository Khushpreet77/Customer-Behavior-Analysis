# Customer-Behavior-Analysis
End  to end data analytic project covering PostgreSql, Python and Power BI


Customer Behavior Analysis Dashboard using SQL, Python & Power BI
  Analyzed customer behavior data to identify purchasing trends, subscription patterns, and revenue insights.
  Used SQL queries to solve business questions and extract meaningful insights from the dataset.
  Performed data cleaning and Exploratory Data Analysis (EDA) using Python libraries such as Pandas, NumPy, Matplotlib, and Seaborn.
  Prepared and transformed data for dashboard visualization and reporting.
  Created an interactive Power BI dashboard with KPIs, slicers, bar charts, and donut charts for customer analysis.
  Visualized customer metrics including average purchase amount, review ratings, sales by category, revenue by age group, and subscription analysis.
  Improved business understanding through data-driven insights and visualization techniques.

  
Skills & Tools Used

SQL, Python, Pandas, NumPy, Matplotlib, Seaborn, Power BI, Data Analysis, Data Visualization, EDA
