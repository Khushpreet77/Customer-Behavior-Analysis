# Customer Behavior Analysis

An end-to-end Data Analytics project that demonstrates the complete workflow of data analysis using **PostgreSQL, Python, and Power BI**. The project focuses on analyzing customer behavior to uncover purchasing trends, subscription patterns, and revenue insights.

---

## Project Overview

This project aims to analyze customer behavior data and generate actionable business insights through data cleaning, exploratory analysis, SQL querying, and interactive dashboard development.

Key objectives include:

- Understanding customer purchasing behavior.
- Identifying subscription trends and customer engagement patterns.
- Analyzing revenue performance across different customer segments.
- Building an interactive dashboard for business decision-making.

---

## Project Workflow

### 1. Data Extraction & Analysis with SQL
- Used PostgreSQL to query and analyze customer data.
- Solved business-related questions using SQL queries.
- Extracted meaningful insights from large datasets.

### 2. Data Cleaning & Exploratory Data Analysis (EDA)
Performed data preprocessing and analysis using Python:

- Data cleaning and handling missing values.
- Exploratory Data Analysis (EDA).
- Statistical analysis and trend identification.
- Data transformation and preparation for visualization.

### 3. Dashboard Development
Built an interactive Power BI dashboard featuring:

- KPI Cards
- Slicers and Filters
- Bar Charts
- Donut Charts
- Customer Segmentation Visuals

---

## Key Insights Visualized

The dashboard provides insights into:

- Average Purchase Amount
- Customer Review Ratings
- Sales by Product Category
- Revenue by Age Group
- Subscription Analysis
- Customer Purchasing Trends
- Revenue Performance Metrics
